## 🔄 Phase 2 — Transfer & Monetize (3-6 เดือน)
> เป้าหมาย: มีรายได้แรก

### 🏠🚗 ระบบโอนทรัพย์สิน (Asset Transfer)
> ขายบ้าน/รถมือสอง พร้อมส่งมอบประวัติการดูแลทั้งหมด

**Flow:**
1. ผู้ขาย กด "โอนทรัพย์สิน" → ระบบสร้าง Transfer Link (หมดอายุ 7 วัน)
2. ผู้ซื้อ เปิด Link → เห็น Preview ประวัติทั้งหมด (ไม่ต้อง login)
3. ผู้ซื้อ กด "รับทรัพย์สิน" → Login → asset + logs ย้ายมาอยู่ใน account ผู้ซื้อ
4. ผู้ขาย → asset หายออกจาก account อัตโนมัติ

**DB ที่ต้องเพิ่ม:**
```sql
asset_transfers (
  id, asset_id, from_user_id,
  token, expires_at,
  status -- pending / accepted / expired
)
```

### 💰 Freemium Model
- **ฟรี:** asset 2 ชิ้น, ไม่มี export PDF
- **Premium ฿99/เดือน:** ไม่จำกัด asset + Export PDF + รูปภาพ
- **One-time ฿299:** Export PDF ต่อ 1 asset (เหมาะกับคนขายรถ/บ้าน)

## 📈 Phase 3 — Revenue & Scale (6-12 เดือน)
> เป้าหมาย: รายได้ stable

### 🛡️ Affiliate ประกันภัย
> ผู้ใช้มีรถ/บ้านอยู่แล้ว = กลุ่มเป้าหมายตรง 100%

**แนวทาง:**
- แสดง Banner "ต่อประกัน" เมื่อใกล้ครบปี (ดึงจาก next_service_date)
- Partner กับ **Roojai, AXA, Rabbit Care** ฯลฯ
- รายได้: **฿500-2,000 ต่อกรมธรรม์** ที่ผู้ใช้คลิกซื้อ

**ข้อดี:**
- ไม่รบกวน UX เพราะแสดงเฉพาะเมื่อ relevant
- Recurring ทุกปีที่ผู้ใช้ต่อประกัน

### 🔧 Partner ร้านซ่อม / อู่รถ
- ผู้ใช้กด "หาช่างซ่อม" → รายชื่อร้านแนะนำในพื้นที่
- ร้านจ่าย **฿100-500 ต่อ booking**

### 📱 ขึ้น App Store / Play Store
- ใช้ Capacitor wrap จาก codebase ปัจจุบัน
- ค่าใช้จ่าย: Apple **$99/ปี** + Google **$25 ครั้งเดียว**
